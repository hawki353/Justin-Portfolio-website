My Data Generator Code

This project was developed to create mock datasets for testing and analysis. By simulating real-world data, I was able to experiment with data processing techniques before handling real company datasets. The generated CSV files were particularly helpful for predictive modeling and visualization tasks.

import random
import pandas as pd

names = ["Stephen", "Mark", "Ben", "Abby", "Emma", "Olivia", "Liam", "Noah", "Sophia", "Mason"]
data = []

for _ in range(500):
    entry = {
        "Name": random.choice(names),
        "Age": random.randint(18, 60),
        "Score": random.randint(50, 100),
        "City": random.choice(["New York", "Los Angeles", "Chicago", "Houston", "Miami"]),
        "Department": random.choice(["Engineering", "Marketing", "Sales", "Finance"])
    }
    data.append(entry)

df = pd.DataFrame(data)
df.to_csv("mock_data.csv", index=False)
print("Mock data saved to mock_data.csv")

My Data Visualization Code

This project focuses on the power of visual storytelling. By using Matplotlib and NumPy, I created an interactive bar chart to visualize random scores. This method allows for a better understanding of data distribution and outlier detection.

import random
import matplotlib.pyplot as plt
import numpy as np

names = ["David", "Antonio", "Nolan", "Ashton", "Bethany", "Lily", "Ethan", "Isabella", "James", "Charlotte"]
scores = [random.randint(50, 100) for _ in names]

plt.figure(figsize=(12, 6))
colors = np.random.rand(len(names), 3)
plt.bar(names, scores, color=colors)

plt.xlabel("Names")
plt.ylabel("Scores")
plt.title("Randomly Generated Scores")
plt.grid(axis='y', linestyle='--', alpha=0.7)

for i, score in enumerate(scores):
    plt.text(i, score + 2, str(score), ha='center', fontsize=12)

plt.show()
